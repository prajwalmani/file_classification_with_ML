File Classification With ML
This is an ML project to clasify files content into a different categories like sports,business and some more 

HOW TO RUN IT:

1. Make a Folder called:

    "Document_Upload"
    
2. Install all the required dependencies:

    flask
    pickle
    sklearn
    filetype
    magic
    nltk
    PyPDF2
    docx
    re
    
3. Run python app.py
